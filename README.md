# Java Jockeys

**Java Jockeys** is a learning-focused organization aimed at helping beginners
understand **Java and core programming concepts** through hands-on practice and
real tooling.

The platform is designed to encourage learning by writing and running actual
Java code, rather than relying only on theory.

One of the core features of Java Jockeys is an **online Java execution
environment**, which allows students to practice Java directly in the browser.

---

## 🎯 Purpose

Java Jockeys was created to:

- Teach Java to beginners in a practical, hands-on way
- Reduce dependency on local IDE and setup for new learners
- Provide a structured platform that can grow into a complete learning system

---

## ✨ Current Features

### 🧭 Dashboard
- Central dashboard representing the Java Jockeys platform
- Organization branding and mission section
- Navigation to available learning tools

### 💻 Online Java Practice Tool
- Web-based interface to write Java programs
- Backend compilation using the real `javac` compiler
- Execute Java programs and view output in the browser

### 🔗 Frontend–Backend Integration
- JavaScript frontend communicates with backend via REST APIs
- Code is sent to the server, compiled, executed, and results are returned

### ⏱ Execution Handling
- Backend controls program execution to avoid server freezing
- Supports basic runtime safety during execution

### 🌐 Public Access for Demo
- Application exposed publicly using a development tunnel
- Enables testing and demonstrations without full deployment
